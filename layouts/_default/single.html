{{ define "main" }} {{/* single.html — Single content page layout This template renders individual content pages. Blog-specific features (metadata, table of contents, post navigation) only appear on pages that are NOT structural content (projects, footer badges). Structural content is identified by its category membership in taxonomy_exclude. Key guards: $is_blog_post — true for non-structural pages with a publication date $is_structural — true for pages in excluded categories (projects, footer) .Params.toc — opt-in table of contents for Markdown posts */}} {{/* Determine if this page is structural content (e.g., project profiles, footer badges). These pages are used for layout composition, not as standalone blog posts. We identify them two ways: 1. Their categories overlap with taxonomy_exclude (via intersect) 2. They set hide_sitemap: true (a convention for embedded content that shouldn't appear as independent pages in search results) */}} {{- $exclude := site.Params.taxonomy_exclude | default slice -}} {{- $is_structural_by_category := and .Params.categories (gt (len (intersect $exclude .Params.categories)) 0) -}} {{- $is_structural := or (.Params.hide_sitemap | default false) $is_structural_by_category -}} {{- $is_blog_post := and (not .Date.IsZero) (not $is_structural) -}} {{/* --- Blog post metadata (date, author, reading time, tags) --- */}} {{ if $is_blog_post }} {{- partial "post-meta.html" . -}} {{ end }} {{/* --- Table of contents (Markdown only, opt-in via front matter) --- */}} {{/* Toph provides unified ToC styling for both Markdown and AsciiDoc. AsciiDoc: Uses its native :toc: document attribute. When the site config sets markup.asciidocExt.preserveTOC: true, Asciidoctor renders the ToC directly inside .Content using div#toc > div#toctitle + ul.sectlevel1 markup. No template action is needed — the theme CSS styles it. Markdown: Opt-in via "toc: true" in front matter. Hugo's .TableOfContents is wrapped in #toc/#toctitle markup to match the AsciiDoc structure, so both formats share the same CSS. We detect AsciiDoc files to avoid rendering a duplicate ToC alongside the one already in .Content. */}} {{ if and .Params.toc (ne .Markup "asciidoc") }} {{ with .TableOfContents }}
