{{/* post-meta.html — Blog post metadata display Shows publication date, last updated date (from git), author, reading time, word count, and linked taxonomy badges (categories and tags) for blog posts. Only included on pages that have a publication date set (guarded in single.html). Key Hugo variables used: .Date — publication date from front matter .Lastmod — last modified date (from git when enableGitInfo: true) .ReadingTime — estimated reading time in minutes .WordCount — total word count of the content .Params.author — optional author override from front matter .Site.Params.biography.name — fallback author from site config .GetTerms "categories" — linked category term pages .GetTerms "tags" — linked tag term pages site.Params.taxonomy_exclude — structural terms to hide (e.g., "footer", "projects") The "last updated" date uses Hugo's .Lastmod, which is automatically populated from git commit history when enableGitInfo: true is set in the site config. It only displays when the last modified date differs from the publication date (i.e., the post has been updated since initial publication). */}} {{- $exclude := site.Params.taxonomy_exclude | default slice -}}
  • Published {{- .Date.Format "January 2, 2006" -}}
    • {{/* Show "Updated" date only when it differs from the publication date. Compares formatted date strings to ignore time-of-day differences. .Lastmod is populated from git history when enableGitInfo: true is set in site config. */}} {{ if and .Lastmod (ne (.Lastmod.Format "2006-01-02") (.Date.Format "2006-01-02")) }}
  • Updated {{- .Lastmod.Format "January 2, 2006" -}}
    • {{ end }}
  • {{- .Param "author" | default .Site.Params.biography.name -}}
  • {{ .ReadingTime }} min read
  • {{ .WordCount }} words
